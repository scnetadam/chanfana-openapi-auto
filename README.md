# 龟钮·自驭 (Guīniǔ Zìyù) — 汽车资讯 AI · 自主价值引擎

> **印信在手，自驭前程。**
> **龟钮铸规，自驭成道。**

**龟钮·自驭** 是基于 X402 微支付协议与 LLM 的汽车垂类 AI 智能体。实时计算全网 KOL/KOC 影响力权重，自动执行价值分账与数据确权。无需人工干预，让优质内容自动获得回报，让车企营销 ROI 清晰可见。

## 🏛️ 品牌文化

自驭意即自动驾驶。契合项目的 AI Agent 内核。

- **字面意**：Self-Driving / Autonomous Control。对应智能驾驶与智能代理。
- **古文意**："虽无舟车，自驭而行"。不需要车夫（中心化平台），车子自己跑，内容自己流转，价值自己分配。
- **项目意**：AI 智能体自动执行。汽车 AI 智能体不需要人工干预，自动抓取 KOL/KOC 数据，自动计算价值，自动通过 X402 发起支付，自动将 Hash 送入数据集市。

### 三驾马车

| 项目 | 角色 | 品牌色 | 逻辑 |
|------|------|--------|------|
| 龟钮·印信 | 手（Action） | 朱红/金石 | 验印为信，授权支付 |
| 龟钮·印证 | 脑（Memory） | 科技蓝/碳灰 | 掌证为权，数据存证 |
| **龟钮·自驭** | 身（Vehicle） | 玄青/赤金 | 持者自驱，自动执行 |

### 价值双螺旋 (VDB)

在「龟钮·自驭」的生态中，每一比特的数据流动都伴随着价值的原子交换。我们不生产内容，我们为内容安装价值引擎；我们不分发流量，我们让流量自我定价。

# 微信小程序TabBar调整说明

## 问题描述

微信小程序报错：
```
app.json: ["tabBar"]["list"] 不能超过 5 项
```

## 原因分析

**平台差异**：
- 支付宝小程序：TabBar最多支持5+项
- 微信小程序：TabBar最多支持5项

## 解决方案

### 调整前（6个Tab）

```
1. 活动
2. 视频
3. 任务
4. AI评估
5. 结算 ← 移除
6. 我的
```

### 调整后（5个Tab）

```
1. 活动
2. 视频
3. 任务
4. AI评估
5. 我的
```

## 访问方式调整

### 结算页面访问

"结算"Tab已移除，可通过以下方式访问：

1. **从"我的"页面访问**
   - 我的钱包 → 查看交易明细

2. **直接路径访问**
   - 页面路径：`pages/settlement/index`
   - 可从其他页面跳转

3. **从商家后台访问**
   - 商家工作台 → 结算管理

## 功能影响

### 不受影响的功能

✅ 所有30个页面均可正常访问
✅ 结算功能完整保留
✅ 商家后台结算管理正常
✅ 钱包页交易流水正常

### 受影响的功能

⚠️ 结算页面无底部Tab入口
⚠️ 需通过其他路径访问结算页

## 替代方案

### 方案一：从"我的"页面添加入口

在钱包页面添加"结算中心"入口按钮。

### 方案二：使用页面跳转

从其他页面添加跳转按钮：
```javascript
uni.navigateTo({ url: '/pages/settlement/index' })
```

### 方案三：合并到"我的"页面

将结算功能整合到钱包页面中。

## 验证结果

✅ Tab数量：5个
✅ 符合微信小程序限制
✅ 核心功能不受影响
✅ 所有页面可正常访问

## 建议

对于需要6个以上Tab的场景，建议：

1. **优先级排序**：将最常用的5个功能放在Tab
2. **次要功能**：通过页面内入口访问
3. **合并功能**：将相关功能合并到一个页面
4. **平台适配**：根据平台特性调整Tab配置

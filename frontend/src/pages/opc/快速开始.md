# OPC创业广场 - 快速开始指南

## 🎯 模块定位

OPC创业广场是一个AI创业扶持平台，为AI创业者提供：
- 🏙️ 按城市划分的创业政策
- 💰 投资补贴申请
- 🎫 TOKEN生态激励
- 📊 创业项目展示
- 🤝 资源对接服务

## 🚀 快速体验

### 1. 访问首页

```javascript
// 从任意页面跳转
uni.navigateTo({ url: '/pages/opc/index' });
```

### 2. 查看城市详情

```javascript
// 查看北京的政策和项目
uni.navigateTo({ url: '/pages/opc/city?id=1&name=北京' });
```

### 3. 申请补贴

```javascript
// 申请某个政策补贴
uni.navigateTo({ url: '/pages/opc/subsidy?policyId=1&city=北京' });
```

## 📱 页面导航

```
首页 (index)
  ├─> 城市详情 (city)
  │     ├─> 政策详情 (policy)
  │     │     └─> 补贴申请 (subsidy)
  │     └─> 项目详情 (project)
  ├─> 项目展示 (project)
  └─> 申请入驻 (apply)
```

## 🎨 核心功能

### 1. 城市政策浏览

**首页展示**:
- 热门城市（北京、上海、深圳）
- 热门政策
- 全部城市列表

**城市详情**:
- 扶持政策列表
- 创业项目列表
- 申请入口

### 2. 政策详情查看

**包含内容**:
- 政策说明
- 扶持内容
- 申请条件
- 申请材料
- 办理流程
- 成功案例

### 3. 补贴申请流程

**步骤**:
1. 填写企业信息
2. 填写项目信息
3. 上传附件材料
4. 提交申请

**所需材料**:
- 企业营业执照
- 财务审计报告
- 项目计划书
- 知识产权证明
- 团队成员证明

### 4. 项目展示

**展示内容**:
- 项目名称和简介
- 公司信息
- 融资情况
- 团队规模
- 创始人背景
- 增长指标

**筛选条件**:
- 全部项目
- 热门项目
- 最新项目
- 融资中项目

### 5. 入驻申请

**申请流程**:
1. 企业信息（公司名称、信用代码、城市）
2. 项目信息（类型、名称、简介、团队）
3. 联系方式（姓名、电话、邮箱、背景）
4. 期望支持（投资、场地、人才、TOKEN等）

## 📝 开发建议

### 1. 数据接入
```typescript
// 替换mock数据为真实API
import { getOPCCities, getOPCPolicies } from '@/api/opc';

const cities = ref([]);
onLoad(async () => {
  cities.value = await getOPCCities();
});
```

### 2. 状态管理
```typescript
// 使用Pinia管理OPC状态
import { useOPCStore } from '@/stores/opc';

const opcStore = useOPCStore();
const { cities, policies } = storeToRefs(opcStore);
```

### 3. 表单验证
```typescript
// 添加更严格的验证
import { validateCreditCode, validatePhone } from '@/utils/validate';

if (!validateCreditCode(formData.creditCode)) {
  uni.showToast({ title: '信用代码格式错误', icon: 'none' });
  return;
}
```

### 4. 文件上传
```typescript
// 接入真实文件上传
import { uploadFile } from '@/api/upload';

const url = await uploadFile(file.path);
formData.attachments.push(url);
```

# 微信小程序配置修复说明

## 问题描述

微信小程序报错：
```
app.json 文件内容错误] app.json: tabBar.list 不能为空
```

## 问题原因

支付宝小程序和微信小程序的 tabBar 配置格式不同：

**支付宝小程序格式**：
```json
{
  "tabBar": {
    "items": [
      {
        "pagePath": "pages/activity/index",
        "name": "活动",
        "icon": "static/images/tab-activity.png",
        "activeIcon": "static/images/tab-activity-active.png"
      }
    ]
  }
}
```

**微信小程序格式**：
```json
{
  "tabBar": {
    "list": [
      {
        "pagePath": "pages/activity/index",
        "text": "活动",
        "iconPath": "static/images/tab-activity.png",
        "selectedIconPath": "static/images/tab-activity-active.png"
      }
    ]
  }
}
```

## 主要差异

| 项目 | 支付宝 | 微信 |
|------|--------|------|
| Tab列表字段 | `items` | `list` |
| 文字字段 | `name` | `text` |
| 图标字段 | `icon` | `iconPath` |
| 选中图标 | `activeIcon` | `selectedIconPath` |
| 文字颜色 | `textColor` | `color` |
| 选中颜色 | `selectedColor` | `selectedColor` |

## 已修复内容

✅ 修改 `tabBar.items` → `tabBar.list`
✅ 修改 `name` → `text`
✅ 修改 `icon` → `iconPath`
✅ 修改 `activeIcon` → `selectedIconPath`
✅ 修改 `textColor` → `color`
✅ 添加 `borderStyle: "white"`
✅ 添加 `sitemap.json` 文件
✅ 添加 `style: "v2"` 配置

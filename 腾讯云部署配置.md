## 🔧 部署步骤

### 1. 准备工作

#### 1.1 安装云开发CLI
```bash
npm install -g @cloudbase/cli
```

#### 1.2 登录云开发
```bash
tcb login
```

### 2. 构建支付宝小程序

```bash
cd D:/X402-DEVECO/frontend
npm install
npm run build:mp-alipay
```

构建产物：`dist/build/mp-alipay`

### 3. 部署后端服务

#### 3.1 创建云托管服务
```bash
tcb hosting deploy -e d1g9iojop685ea11a
```

#### 3.2 配置环境变量
在云托管控制台设置：
- `JWT_SECRET` - JWT密钥
- `GLM_API_KEY` - 智谱AI密钥
- `COS_SECRET_ID` - 腾讯云COS密钥ID
- `COS_SECRET_KEY` - 腾讯云COS密钥
- `ALIPAY_APPID` - 支付宝AppID
- `ALIPAY_PRIVATE_KEY` - 支付宝私钥

### 4. 配置域名

#### 4.1 绑定自定义域名
在云托管控制台绑定域名：
- `guiniu-xin-281816-7-1306394233.sh.run.tcloudbase.com`

#### 4.2 配置HTTPS
云托管自动提供HTTPS证书

### 5. 部署前端

#### 5.1 上传小程序代码
使用支付宝小程序开发者工具：
1. 导入项目：`D:/X402-DEVECO/frontend/dist/build/mp-alipay`
2. 配置AppID：2021006168648698
3. 上传代码
4. 提交审核

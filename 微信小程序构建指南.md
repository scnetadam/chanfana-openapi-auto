# 微信小程序构建指南

## 问题说明

当前项目使用 uni-app Vue3 版本构建微信小程序时遇到依赖兼容问题：
```
"isInSSRComponentSetup" is not exported by "vue.runtime.esm-bundler.js"
```

这是 uni-app 5.14 版本与 Vue 3.4+ 的已知兼容问题。

## 解决方案

### 方案一：使用 HBuilderX 构建（推荐）

1. 下载安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)
2. 打开 HBuilderX，导入项目 `D:\X402-DEVECO\frontend`
3. 点击 `发行` → `小程序-微信`
4. 等待构建完成
5. 使用微信开发者工具导入构建产物

### 方案二：降级 Vue 版本

```bash
cd frontend
npm install vue@3.3.4 --save
npm run build:mp-weixin
```

### 方案三：使用支付宝小程序代码

微信小程序和支付宝小程序代码完全相同，可以：

1. 在微信开发者工具中直接导入 `dist/build/mp-alipay`
2. 修改 `project.config.json` 中的 appid

```json
{
  "appid": "wx85f61ef6c155f30d",
  "projectname": "x402-weixin"
}
```

## 常见问题

### Q: 为什么微信小程序构建失败？
A: uni-app Vue3 版本与微信小程序编译器存在兼容问题，建议使用 HBuilderX 或使用支付宝小程序产物。

### Q: 可以直接用支付宝小程序的代码吗？
A: 可以！uni-app 编译的代码在支付宝和微信小程序中基本相同，只需修改 appid 配置。

### Q: 如何调试微信小程序？
A: 使用微信开发者工具导入构建产物，可以实时预览和调试。
